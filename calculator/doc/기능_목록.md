# 계산기 기능 목록

## 입출력

### 입력

- [x] 메뉴 번호를 입력받는다.
- [ ] 계산 시, 중위 연산식을 입력받는다.

### 출력

- [ ] 메뉴 정보를 출력한다.
- [ ] 조회 시, 조회된 정보 출력한다.
- [ ] 계산 시, 계산 결과 출력한다.

## 저장

- [ ] 중위 연산식을 저장한다.
- [ ] 계산 결과를 저장한다.

## 연산

- [ ] 덧셈
- [ ] 곱셈
- [ ] 뺄셈
- [ ] 나눗셈

## 계산

- [ ] 중위 연산식 계산

## 예외처리

- [ ] Overflow 발생시, throw new ArithmeticException("계산 중 Overflow가 발생했습니다.")
- [ ] 계산 시 입력이 중위 연산식이 아닐 경우, throw new IllegalArgumentException("중위 연산식이 아닙니다.")
- [ ] 0으로 나눌 경우, throw new IllegalArgumentException("0으로 나눌 수 없습니다.")
