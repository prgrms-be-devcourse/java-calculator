## OOP적 계산기 구현

## 요구사항

## domain
### 계산기
- (상태) String 계산식
- 덧셈 기능
- 뺄셈 기능
- 곱셈 가능
- 나눗셈 기능

### 기록
- (상태) LinkedHashMap<계산식, 값> 기록
- 저장 기능
- 반환 가능

## View
### 입력
- 버튼을 입력하는 기능
  - (예외) (정규표현식) 1또는 2가 아닌 경우
- 연산을 입력하는 기능
  - (예외) (정규표현식) 숫자와 연산이 아닌 경우

### 출력
- 조회, 계산 버튼 출력 기능
- 연산 결과를 출력하는 기능
- 연산 기록을 출력하는 기능
